# GSSoC Contribution Guide

This repo is prepared to accept contributions for GSSoC 2026. Maintainers: please add the following to help students get started:

- Mark beginner-friendly issues with the `good first issue` label.
- Provide clear setup steps in `CONTRIBUTING.md` and small, well-scoped tasks.
- Add `help wanted` labels to issues suitable for volunteers.
- Add a `mentor` contact in issues or project board.

If you want to apply the `GSSoC` tag or require tasks to be reserved, add instructions here.
